# My Portfolio Website

## Description
This is a personal portfolio website for Surajit Mondal, showcasing his projects, skills, and experience as a Full Stack Web Developer. It serves as a digital resume and a demonstration of his capabilities in modern web technologies, built with a focus on clean UI, performance, and practical problem-solving.

## Features
-   **Hero Section:** An engaging introductory section highlighting Surajit Mondal's role as a Full Stack Web Developer.
-   **About Me Section:** Provides an introduction to his background, education (MCA), and web development experience (1+ years, 5+ projects).
-   **Skills Section:** Details his technical proficiencies across Frontend (HTML, CSS, JavaScript, React, Next.js, Tailwind CSS), Backend (Node.js, Express.js, REST APIs, MongoDB, Appwrite), and Tools & Workflow (Git & GitHub, Postman, Vercel).
-   **Project Showcase:** Features detailed views of projects including:
    *   E-commerce Platform (TechStore)
    *   Doctor Appointment Booking (Medicure)
    *   Portfolio Website (This project)
-   **Contact Section:** Includes contact information (email) and links to social profiles (LinkedIn, GitHub, X).
-   **Responsive Design:** Optimized for various devices.

## Technologies Used
-   **Frontend:** Next.js, React, TypeScript, HTML, CSS, JavaScript
-   **Styling:** Tailwind CSS
-   **Backend:** Node.js, Express.js, REST APIs, MongoDB, Appwrite
-   **Tools & Workflow:** Git, GitHub, Postman, Vercel
-   **Icons:** Tabler Icons (`@tabler/icons-react`)

## Getting Started

### Prerequisites
-   Node.js (version X.X.X or higher)
-   npm or yarn

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/your-portfolio.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd your-portfolio
    ```
3.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

### Running the Development Server
```bash
npm run dev
# or
yarn dev
```
Open [https://portfolio-sm-seven.vercel.app/](https://portfolio-sm-seven.vercel.app/) with your browser to see the result.

## Deployment
This project is deployed on Vercel. For deployment details, refer to the Next.js deployment documentation.
